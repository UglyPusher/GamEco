# Core

Базовые интерфейсы и реализации, лежащие в основе архитектуры симуляции.

## Содержание

| Файл                          | Назначение                                                                 |
|------------------------------|----------------------------------------------------------------------------|
| `IIdentifiable.h`            | Интерфейс объектов с уникальным идентификатором.                          |
| `BaseIdentifiable.h/.cpp`   | Базовая реализация `IIdentifiable` с автоинкрементным ID.                |
| `IStepable.h`                | Интерфейс объектов, обновляемых каждый шаг симуляции.                     |
| `BaseStepable.h/.cpp`       | Базовая реализация `IStepable` с коллекцией потомков и перемешиванием.   |
| `Random.h/.cpp`             | Централизованный генератор случайных чисел `std::mt19937`.                |

## Основные принципы

- **Уникальность:** Все объекты, реализующие `IIdentifiable`, получают уникальный `id`.
- **Обновляемость:** `IStepable` позволяет объекту участвовать в тактах симуляции. `BaseStepable` управляет дочерними stepable-объектами.
- **Нормализация порядка:** В `BaseStepable::step()` коллекция потомков перемешивается перед вызовами `step()`, исключая привилегии порядка добавления.
- **Контроль воспроизводимости:** Используется глобальный генератор случайных чисел с возможностью задания seed.
